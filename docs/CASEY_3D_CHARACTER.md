# Casey 3D Character

## Status
First-pass stylized 3D bust generated in Blender.

## Important Scope
This is a portrait-inspired stylized 3D character, not a photogrammetry scan or exact likeness-verified avatar. A single front-facing image is enough for visual cues, but not enough for identity-accurate 3D reconstruction.

## Source Cues Used
- Floating head/bust composition.
- Warm skin material.
- Close-cropped hair.
- Blue sunglasses with gold frame.
- Small goatee.
- Bright blue sky and soft cloud environment.

## Privacy Handling
The exported `.blend` and `.glb` do not embed the original portrait as a hidden texture or reference plane.

## Outputs
- Render: `outputs/casey_3d_character/casey_3d_bust_render.png`
- Editable Blender file: `outputs/casey_3d_character/casey_3d_bust.blend`
- Web-ready 3D asset: `outputs/casey_3d_character/casey_3d_bust.glb`

## Next Refinement Options
- Sculpt stronger likeness in Blender from additional reference angles.
- Improve facial proportions and expression.
- Build a cleaner hero-ready GLB with lower polygon count and baked materials.
- Replace the current procedural bust with a dedicated character pipeline if exact likeness matters.

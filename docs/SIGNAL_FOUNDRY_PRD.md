# BattleLabs.co Homepage PRD

## Goal

Build a cinematic, founder-facing BattleLabs.co homepage that feels like a complete applied AI lab / founder operating system company site.

## Required Stack

- React
- Tailwind CSS
- Framer Motion
- Google Fonts: Instrument Serif and Inter Tight

## Required Assets

- Hero poster: `https://playground.bravebrand.com/assets/backgrounds/signal-foundry-painted-city-hero.webp`
- Hero video: `https://d8j0ntlcm91z4.cloudfront.net/user_30c6yRkxUog0TZ5432rCR7HN4Pe/hf_20260501_062927_2b8ce586-f555-4610-88ae-b2d3752ede3b.mp4`
- Sky-garden poster: `https://playground.bravebrand.com/assets/backgrounds/signal-foundry-sky-garden.webp`
- Sky-garden video: `https://d8j0ntlcm91z4.cloudfront.net/user_30c6yRkxUog0TZ5432rCR7HN4Pe/hf_20260501_082435_42398084-1c8d-48e5-a962-fe7c28c124e6.mp4`
- Pixel flower: `https://playground.bravebrand.com/assets/backgrounds/signal-foundry-pixel-flower.webp`

## Sections

1. Full-screen video hero with glass navigation and lower-left glass hero card.
2. Sky-garden support section with video inside a rounded image frame.
3. Manifesto section with pixel flower and editorial serif statement.
4. System section with four product cards.
5. Dark cinematic closing CTA with footer.

## Acceptance Criteria

- Hero video fills the viewport and uses the painted-city poster fallback.
- Sky-garden video plays inside the rounded support image frame without white bars.
- Pixel flower appears in the manifesto section.
- Editorial headings use Instrument Serif.
- Body, navigation, cards, labels, and CTAs use Inter Tight.
- Motion uses Framer Motion, respects reduced motion, and avoids generic AI gradients or decorative blobs.
- Page reads as a complete company homepage, not a single hero section.
